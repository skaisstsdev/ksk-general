Глобальные правила
Применяются на всех страницах
Hero desktop: 2 колонки — текст слева, фото 4:5 справа без рамки. Badge "Seit 2013" снизу-слева от фото.
Hero mobile: текст сверху → кнопка → фото с отрицательным margin перекрывает кнопку снизу. z-index кнопки выше.
Кнопки — 3 вида: Primary (фиолет) · Outline (вторичная) · Liquid Glass (на фиолетовом CTA-фоне). 48px lg, 36px md. Border-radius: full.
Фото — единый формат: 4:5 для Hero · 4:3 для карточек · 1:1 для команды. object-fit: cover везде. border-radius: 12px.
Язык: floating pill зафиксирован в левом нижнем углу экрана — всегда видим.
Акцентный текст после Hero: editorial-абзац с ключевыми данными компании — без рамок, border-left accent, вместо stats bar.

Мобильная версия
Глобальные правила мобильной адаптации
Hero — всегда: текст сверху → кнопка (100% ширина) → фото с отрицательным margin-top перекрывает низ кнопки. z-index кнопки выше фото — кликабельна.
Grid-3/4 → snap-scroll: карточки горизонтальный overflow с snap. Видна одна карточка + край следующей. Не вертикальный стек.
Grid-2 / Split → стек: фото сверху, текст под ним. Исключение: kontakt.html — форма идёт после текста.
CTA кнопки → 100% ширина, стек вертикально. Минимальная высота touch target: 44px.
Stats row → 2×2 (четыре ячейки в две строки по две).
Footer → 1 колонка с разделителями. Accordion для сворачивания секций на маленьких экранах.
Навигация: hamburger → fullscreen overlay с крупными serif ссылками. Закрывается свайпом вправо или ✕.
Якорная навигация (Leistungen) → горизонтальный overflow-scroll, sticky под header.
FAQ поиск + фильтр: поиск на всю ширину, категории — горизонтальный скролл.
Формы: все поля 100% ширина, label над input. Wizard — одна панель на весь экран. Submit 100%.
Google Maps: высота уменьшается с 400px до 220px. Touch-friendly.
Leitbild 5 пунктов → 1 колонка вертикальный стек. Команда 3 карточки → горизонтальный список с аватаром слева.
